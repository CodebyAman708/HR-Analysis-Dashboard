# HR Analytics Dashboard | Power BI

## Project Overview

This project focuses on analyzing employee data to identify key factors influencing employee attrition, workforce demographics, and overall organizational performance. The dashboard was developed using Microsoft Power BI to provide actionable insights for HR decision-making.

## Objectives

* Analyze employee attrition trends
* Identify factors contributing to employee turnover
* Monitor workforce demographics
* Evaluate employee satisfaction and engagement metrics
* Support data-driven HR strategies

## Tools & Technologies

* Microsoft Power BI
* Power Query
* Data Modeling
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation

## Key Insights

* Analyzed employee attrition across different departments.
* Identified job roles with the highest turnover rates.
* Examined the impact of age, salary, education, and experience on employee retention.
* Evaluated employee satisfaction and work-life balance metrics.
* Created interactive visualizations for effective workforce analysis.

## Dashboard Features

* Employee Attrition Analysis
* Department-wise Employee Distribution
* Gender Diversity Analysis
* Age Group Segmentation
* Job Role Performance Metrics
* Employee Satisfaction Insights
* Interactive Filters and Slicers

## Project Outcomes

The dashboard enables HR teams to identify attrition patterns, understand workforce demographics, and make informed decisions to improve employee retention and organizational performance.
